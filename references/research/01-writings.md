## 四、核心论点（≥3次 = 真信念）

### 4.1 量化轮动 / 满仓不择时
- 出现：所有书籍+几乎所有长文+所有访谈
- 表述："全资产满仓量化轮动""大轮套小轮"
- 来源：说股市系列十一(1)(2)、四大金刚系列、播客、《寻找鱼多的池塘》全书主线

### 4.2 用广度替代深度
- 出现：《寻找鱼多的池塘》全书主题、雪球访谈、雪球嘉年华
- 表述："宽度重于深度""选择大于努力""去鱼多的池塘钓鱼"
- 多品种轮动（封基/分离债/分级A/可转债/债券）

### 4.3 封基折价率策略
- 网名由来：2007年全仓封基获180%收益
- 关键公式：修正年化折价率 = (折价率 - 1.5%) / 剩余年限
- 折价率在牛市起"加速器"作用，熊市起"安全垫"作用

### 4.4 BIAS乖离率作为核心轮动指标
- BIAS4.0是其"看家指标"
- 三线四区域：分为> A1 / A2~A1 / A3~A2 / < A3，分别对应空仓/满仓/空仓/满仓

### 4.5 四大金刚
- 封基、债券（分离债→Q债）、分级A（2020年退出）、可转债（2021年后主力）
- 四个非股票小众品种的统称

### 4.6 可转债涨跌不对称性 / 满仓轮动
- 2021年清仓所有白马股全换可转债
- "正股涨1%转债涨约0.6%，正股跌1%转债跌约0.4%"
- 在果仁网跑100+策略，花一个月用Excel回测排除BUG

### 4.7 守拙比取巧更重要
- "做人要有情有义，但投资要无情无义"
- 接受"中位数人"定位，不追求战胜市场的秘籍

### 4.8 高股息率因子优于高ROE因子
- 回测结论：加入高ROE因子反而导致收益率下滑

### 4.9 小市值因子（卫星策略）
- 选择市值最低20只股票，剔除EP<0和扣非ROE<-1
- 回测2007-2022年250+倍（年化约42%），但最大回撤70%+

## 五、自创术语

| 术语 | 含义 |
|------|------|
| **四大金刚** | 封基、债券、分级A、可转债四个品种统称 |
| **不辣/微辣** | 可转债策略按频率和风险分级（火锅辣度类比） |
| **三线四区域** | BIAS4.0的四个行情分区 |
| **大轮套小轮** | 大类资产间BIAS轮动 + 内部最优因子轮动 |
| **鱼多的池塘** | 流动性好/机会多/散户有优势的市场 |
| **折价率安全垫/加速器** | 封基折价在熊市的保护和牛市的加倍作用 |
| **修正年化折价率** | (折价率 - 1.5%) / 剩余年限 |
| **脉冲卖出** | 可转债突然大涨时果断卖出，不等轮动周期 |
